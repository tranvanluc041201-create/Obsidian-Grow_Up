图像联动区

现在这版专门收紧了排布，目的不是好看，而是让你拖滑块时不用上下找图。

顶点
(0.00, 0.00)
这是图像的最低点或最高点。看顶点的位置，比背公式更容易把平移感建立起来。
开口方向
向上
`a > 0` 向上，`a < 0` 向下。这个判断必须形成反射，不要每次都重新想。
宽窄感
标准
`|a|` 越大越窄，越靠近 0 越宽。这个是很多人只会背但没有图像感觉的地方。
