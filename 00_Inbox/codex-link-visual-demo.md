# 动态视觉图链接测试

这篇只测试一种更轻的方式：**笔记里放链接，点击后单独打开 demo**。

## 入口

- [打开函数变换交互 demo（浏览器）](http://127.0.0.1:8765/00_Inbox/codex-function-demo.html?v=20260311b)

## 你要观察什么

1. 点击后是不是直接打开完整 demo
2. 是不是没有 `iframe` 的高度与嵌入刷新问题
3. 这种方式对你来说，会不会比内嵌更顺手

## 结论怎么判断

- 如果你更在意稳定和省维护，链接更优
- 如果你更在意边看笔记边拖动，再考虑内嵌
