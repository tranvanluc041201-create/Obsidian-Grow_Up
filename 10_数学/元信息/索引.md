## 统计

### 按课程统计错题

```dataview
TABLE length(rows) AS 数量
FROM "10_数学"
WHERE type = "mistake"
GROUP BY course
SORT length(rows) DESC
```

### 按错误类型统计

```dataview
TABLE length(rows) AS 数量
FROM "10_数学"
WHERE type = "mistake"
GROUP BY error_type
SORT length(rows) DESC
```
