# Quick Start - Physics

*每日操作流程*

---

## Morning

1. Open `inbox/YYYY-MM-DD.md`
2. Set goals:
   - [ ] Mechanics: ___
   - [ ] Electromagnetism: ___
3. Set diagram targets:
   - Scene: ___  FBD: ___  Field: ___  Loop: ___

---

## When Solving

### Step 1: Draw Diagram (20-60s)
- Mechanics → FBD
- Kinematics → Scene
- E&M → Field
- Circuits → Loop

### Stuck >5min?
Record and continue:
```
- Mech 3.5 - stuck at system choice - E2?
```

### Wrong answer?
Note error type (E1-E7):
```
- Elec 2.3 - E4 used wrong principle
```

---

## Evening (15min)

Move items to folders:

| Item | Destination | Template |
|------|-------------|----------|
| Problem | `mechanics/problems/` | phys_problem |
| Mistake | `mechanics/mistakes/` | phys_mistake |
| Concept | `mechanics/concepts/` | phys_concept |

Update diagram count in inbox.

---

## Sunday

1. Create weekly note
2. Review diagram statistics
3. Analyze E1-E7 distribution
4. Set next week's focus
