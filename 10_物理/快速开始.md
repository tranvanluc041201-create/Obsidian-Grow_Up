# 快速开始 - 物理学习流程

*7块协议 + Pattern 训练*

---

## 视频材料输入

> [!tip]
> 默认先发：`视频链接 + 一句主线` 或 `一个卡点`。
> 教材 / 讲义默认**按需读取**，不默认整份展开。
> 只有在标准定义、图像板书、条件近似或正式笔记需要时，再按目录去读对应位置。
> 物理视频流里，教材只在 `预热`、`纠偏`、`定稿` 三个时机插入。

---

## 教材触发时机

1. `预热`
   只在新模型、新场景、新符号较多，或先修概念明显不足时使用。
   默认只看 `3~5` 分钟：对象定义、图示、条件与近似。
2. `纠偏`
   先看完视频并做一轮图像重构。
   如果对象、近似条件、适用范围或标准推导仍不稳，再查教材对应 `1~3` 页。
3. `定稿`
   只有在正式笔记前，才用教材统一图像、条件表述和标准公式。

---

## 解题 7 块协议

每道题必须完整走完这7步：

```
1. Goal      → 明确目标：求什么？
2. 微图解    → 画图：Scene/FBD/Field/Loop
3. 近似      → 列出简化假设
4. 原理      → 选择物理原理，列方程
5. 计算      → 关键代数步骤
6. 检查      → 量纲/极限/数量级
7. 输出      → 生成关联卡片
```

---

## 每日流程

### 上午：新题学习 (60分钟)
- 选1道代表性题目
- 完整走7块协议
- 生成概念卡 + 题目卡

### 下午：Pattern 训练 (40分钟)
- 同类题3道
- 计时完成
- 记录用时和正确率

