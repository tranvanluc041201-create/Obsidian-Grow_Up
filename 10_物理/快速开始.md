# 快速开始 - 物理学习流程

*7块协议 + Pattern 训练*

---

## 解题 7 块协议

每道题必须完整走完这7步：

```
1. Goal      → 明确目标：求什么？
2. 微图解    → 画图：Scene/FBD/Field/Loop
3. 近似      → 列出简化假设
4. 原理      → 选择物理原理，列方程
5. 计算      → 关键代数步骤
6. 检查      → 量纲/极限/数量级
7. 输出      → 生成关联卡片
```

---

## 每日流程

### 上午：新题学习 (60分钟)
- 选1道代表性题目
- 完整走7块协议
- 生成概念卡 + 题目卡

### 下午：Pattern 训练 (40分钟)
- 同类题3道
- 计时完成
- 记录用时和正确率

## 卡片模板

### 概念卡
在对应学科概念文件夹创建：
```markdown
# 概念 - XXX

## 一句话定义

## 数学表达

## 适用条件

## 典型应用

## 常见误区
```

### 错题卡
在对应学科错题文件夹创建：
```markdown
# 错误 - YYYYMMDD

## 错误类型
E1/E2/E3/E4/E5/E6/E7

## 错误分析

## 正确做法

## 预防措施
```
