# SITO 单电感三输出 Buck｜闭环复现总览

---

## 一、这次实验到底做成了什么

- **目标**：把单电感三输出 Buck 的三路闭环控制跑稳，并尽量复现论文中的稳态效果
- **结果**：三路输出最终可以稳定在目标附近：
  - `V1 ≈ 12V`
  - `V2 ≈ 5V`
  - `V3 ≈ 3V`
- **现象**：启动阶段仍会出现超调、下探、振铃，但随后可以收敛到稳定状态
- **结论**：就“闭环复现成功”这个目标来说，这次实验已经跑通了主链路

## 二、我认为这次实验最重要的收获

### 1. 闭环不是“PI 一接上就行”

真正决定系统能不能工作的，是整条控制链路是否一致：

`误差 e = Vref - Vfb → PI → Limiter → ramp 比较 → PWM → 开关时序`

只要其中有一环量纲不对、极性反了、参考点错了，整个系统就会表现得像“控制器失灵”。

### 2. 你现在已经有一套自己的排错优先级

这次实验中，最有价值的不是某个参数，而是你已经摸索出一条很实用的排错顺序：

**离谱值 → 参考点 → Vgs / 开关模型 → 量纲 / 极性 → PI 参数**

这条顺序以后还能继续复用。

### 3. 你现在的学习方式已经很明确

不是先学完整理论再做实验，而是：

**先做项目 → 抽经验 → 再反推理论**

这对你是有效的，因为你已经能把仿真问题抽成方法和错误类型。

## 三、这次最关键的两个坑

| 问题 | 表现 | 根因 | 快速定位 |
|------|------|------|----------|
| MOSFET 接反 / 开关不导通 | 电压或波形表现离谱，输出起不来 | 器件方向或导通条件不满足 | 看电流路径、看 `Vgs` |
| 参考点不一致 | 电压表显示离谱值，采样不可信 | 看起来接地，实际不是同一参考点 | 采样负端硬拉到 `VDC-` 测试 |

## 五、这次实验的验收标准

- [x] 三路输出都能稳定到目标附近
- [x] `PI` 输出与 `Limiter` 输出处在合理范围
- [x] 运行时没有明显数值溢出或报错
- [ ] 进一步用游标读取稳态平均值，替换当前目测值
- [ ] 继续优化启动阶段的超调与下探
