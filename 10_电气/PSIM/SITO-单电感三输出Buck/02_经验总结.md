# SITO 单电感三输出 Buck｜经验总结

*单电感三输出 Buck 变换器｜闭环复现后的可复用结论*

---

## 一、30 秒结论

- **目标**：让三路输出 `12V / 5V / 3V` 在闭环下稳定工作
- **结构**：单电感 + 分时供能 + 每路独立反馈 + 共用载波比较
- **最重要的理解**：每一路最后能拿到多少能量，不只是看 `PI`，更要看开关时序和比较链是否正确

---

## 二、三分钟排错流程

这是你这次项目里最有价值、也最能复用的一部分。

### 第 0 步（30 秒）：先把观测窗口搭对

必须同时看到：

- `Vout1/2/3`
- `Vfb1/2/3`
- `e1/2/3`
- `PIout`
- `Limout`
- `ramp`
- `PWM1/2/3`
- `IL`

**判断标准**：

如果你看不到 `控制量 d` 和 `ramp` 之间的比较关系，就先别改电路，先把观测补齐。

### 第 1 步（60 秒）：先看 Limiter 有没有卡死

重点看三件事：

1. `Limout` 是否在 `0 ~ 1` 范围内
2. `ramp` 是否也是 `0 ~ 1`，且频率正确（`50kHz`）
3. 比较器极性是否正确：`ramp < d → PWM = 1`

如果 `Limout` 长期贴边，优先排查：

- 误差极性
- 限幅范围
- `ramp` 与 `Limiter` 的量纲一致性

### 第 2 步（60 秒）：做“参考点硬拉”验证

操作很简单：

1. 把采样器负端直接拉到 `VDC-`
2. 看输出是否立刻恢复正常

如果立刻恢复，根因大概率就是：

**参考点不一致**

### 第 3 步（30 秒）：确认 `Vgs` 与开关是否真的导通

要看：

```text
Vgs = Vgate - Vsource
```

不是看 `Gate` 对地。

如果 `Vgs` 太小或是负的，那么：

- 开关看起来有 PWM
- 实际上却没真正导通

这通常说明是：

**浮地驱动问题**

## 三、这次实验抽出来的通用认识

### 1. PWM 生成逻辑

控制链本质是：

```text
Vref
→ 与 Vfb 做差
→ PI
→ Limiter
→ 与 ramp 比较
→ 生成 PWM
```

关键要求：

**`Limiter` 输出和 `ramp` 必须在同一量纲范围内。**

### 2. 控制信号流

你可以把它理解成三层：

1. **PI 输出**：告诉系统“该多大力度”
2. **Limiter**：把控制量限制在合理范围
3. **比较器**：把连续值变成开关时序

### 3. 软启动的真正作用

软启动不是为了让波形“好看”，而是为了让参考值不要一步跳满。

形式上可以写成：

```text
Vref_soft = Vref × k(t)
```

其中 `k(t)` 从 `0` 缓慢上升到 `1`。

这样能显著减轻启动阶段的冲击。

## 四、这次最致命的几个坑

| 坑点 | 典型表现 | 根因 | 快速检查 | 修复方向 |
|------|----------|------|----------|----------|
| 参考点不一致 | 输出离谱、采样不可信 | 不同节点被当成同一地 | 采样负端硬拉 `VDC-` | 统一参考点 |
| 高边驱动浮地 | 有 PWM 但输出起不来 | `Source` 浮动导致 `Vgs` 不足 | 直接测 `Vgs` | 修驱动或先用理想开关 |
| Limiter 卡死 | 控制量长期 0 或 1 | 极性错、量纲错、积分饱和 | 同屏看 `e/PIout/Limout/ramp` | 修极性与范围 |
| 续流路径错误 | 输出起不来或异常 | 二极管方向 / 回路错误 | 查关断时电流路径 | 修正方向与路径 |

## 五、你已经形成的信号命名习惯

| 名称 | 真正含义 | 应该参考谁 | 常见错误 |
|------|----------|------------|----------|
| `Vc` | 控制电压 / 占空比指令 | 控制器参考地 | 与 `ramp` 量纲不匹配 |
| `Vgs` | 栅源电压 | `Source` | 误看成 `Gate` 对地 |
| `Vfb` | 反馈电压 | 控制器地 / `VDC-` | 采样参考点错 |
| `Limout` | 限幅输出 | 控制器地 | 长期卡在边界 |
| `d` | 占空比指令 | 数值量 | 比较器极性写反 |

## 六、可复用模块

### 模块 1：误差 + PI + Limiter

- 输入：`Vref_i`、`Vfb_i`
- 输出：`d_i`
- 检查点：稳态时不应长期贴边

### 模块 2：ramp + 比较器 → PWM

- 输入：`d_i`、`ramp`
- 输出：`PWM_i`
- 检查点：`d` 越大，占空比应越大

### 模块 3：开关 + 电感 + 输出级

- 检查重点：
  - 关断时是否有续流路径
  - 电感电流是否连续

## 七、迁移到 LTspice 时要注意什么

### 常见模块映射

| PSIM 模块 | LTspice 对应 | 说明 |
|-----------|--------------|------|
| Voltage Sensor | 电压探针 | 点击节点读取 |
| PI Controller | B-source | 用行为源写表达式 |
| Limiter | limit 函数 | 输出限到 `0 ~ 1` |
| Comparator | B-source 比较 | 用条件表达式生成高低电平 |
| Ramp | `PULSE` | 保证周期和幅值匹配 |
| Controlled Switch | `SW` | 用电压控制开关模型 |
| MOSFET | `nmos` / `pmos` | 需要模型参数 |

### 仿真策略

时域仿真一般写：

```spice
.tran 0 10m 0 1u
```

重点观察：

- 电感纹波
- 输出纹波
- 路间耦合
- 启动瞬态
