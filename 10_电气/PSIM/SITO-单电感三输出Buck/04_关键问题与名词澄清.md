# SITO 单电感三输出 Buck｜关键问题与名词澄清

---

## 一、为什么要单独整理这篇

你这次实验里其实已经碰到了一个很典型的问题：

> 很多问题不是“不会调参数”，而是**名词和参考点没彻底弄清楚**。

尤其是下面几个量，如果不清楚，很容易把测量、驱动和反馈都看错：

- `Vdc`
- `VDC+ / VDC-`
- `Vbs`
- `VB / VS`
- `Vgs`
- `Vfb`

---

## 二、Q1：Vdc 是什么？VDC+ / VDC- 又是什么？

### 1. VDC+ / VDC- 是节点名

它们表示的是：

- `VDC+`：直流母线正端
- `VDC-`：直流母线负端

也就是说，它们首先是**电路中的两个节点**，不是一个单独的“电压值”。

### 2. Vdc 是这两个节点之间的电压差

所以：

```text
Vdc = V(VDC+, VDC-)
```

如果你的输入电源是 100V，那么通常就是：

```text
Vdc ≈ 100V
```

### 一句话记忆

**VDC+ / VDC- 是“位置”，Vdc 是“这两个位置之间的电压差”。**

---

## 三、Q2：Vbs 是什么？

`Vbs` 是 **自举供电电压**。

它专门服务于高边驱动，因为高边 `NMOS` 的 `Source` 不是固定接地，而是跟着开关节点一起上下浮动。

所以高边驱动不能简单看：

```text
Gate 对地是多少伏
```

真正该看的是：

```text
Vbs = VB - VS
```

其中：

- `VB`：自举电容上端
- `VS`：高边 MOSFET 的 `Source`，也就是开关节点

### 一句话记忆

**Vbs 是高边驱动自己的“小电池电压”，必须看 `VB - VS`，不是看 `VB` 对地。**

---

## 四、Q3：MOSFET 的 Source 是“输出端”吗？

不是。

MOSFET 三端是：

- `Gate`
- `Drain`
- `Source`

其中 `Source` 最重要的意义不是“输出端”，而是：

> 它是门极驱动的参考点。

所以判断 MOSFET 是否真正导通，不能看：

```text
Gate 对地
```

而要看：

```text
Vgs = Vgate - Vsource
```

### 一句话记忆

**开不开门看 `Vgs`，不是看 `Gate` 对地。**

---

## 五、Q4：Vfb 是什么？

`Vfb` 是 **反馈信号**。

它代表的是：

> 控制器眼里“当前输出到底是多少”

所以：

- `Vref` 是你希望它达到的目标
- `Vfb` 是控制器实际看到的结果

误差就是：

```text
e = Vref - Vfb
```

如果 `Vfb` 的参考点错了，即使实际输出对了，控制器也会“以为自己没对”。

### 一句话记忆

**Vfb 不是单纯的测量值，而是控制器拿来闭环比较的那条反馈信号。**

---

## 六、这几个量最容易混淆的点

| 名称 | 真正含义 | 最容易犯的错 |
|------|----------|--------------|
| `Vdc` | 直流母线电压 | 把节点名和电压值混为一谈 |
| `VDC+ / VDC-` | 母线正负节点 | 以为它们本身就是“电压” |
| `Vbs` | 高边驱动的自举电压 | 看成对地电压 |
| `Vgs` | 栅源电压 | 错看成 `Gate` 对地 |
| `Vfb` | 控制器看到的反馈电压 | 忽略参考点 |

---

## 七、和这次实验最相关的判断

### 判断 1

如果高边 `Source` 在浮动，那么：

**`Gate` 对地看起来有电压，不代表开关真的导通。**

### 判断 2

如果反馈信号参考点不一致，那么：

**控制器会在错误的数据上闭环。**

### 判断 3

如果连 `Vdc / Vgs / Vfb / Vbs` 都混了，后面所有调试都会变慢。
