# 数据类型 / 表达式

## 表达式抓手
- 看见 关系表达式
    - 结果只会是真或假
- 看见 条件表达式
    - 条件 ? 真时取前 : 假时取后
- 看见 逗号表达式
    - 整体值看最后一个

## 两组核心对比
- 'a' vs "a"
    - 一个是字符
    - 一个是字符串
- 5 / 2 vs 5 / 2.0
    - 一个结果是 2
    - 一个结果是 2.5
- 0 vs '0'
    - 一个是数值 0
    - 一个是字符 0

## 候选易混点
- 0 < a < 2
    - 不是数学连写
    - 先算 0 < a
    - 再拿结果 0 或 1 去和 2 比
- a = (3, 5)
    - 前面会算
    - 最后结果看 5
- (double)a / b
    - 先把 a 转成 double
    - 再去做除法

## 做题抓手
- 先看参与运算的是谁
- 再看类型会不会变
- 再看是不是整除
- 最后看整个表达式的值是谁
