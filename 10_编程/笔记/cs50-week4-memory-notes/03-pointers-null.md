# 3）指针与 NULL

## 🎯 一句话结论

> 指针里放的是地址；`NULL` 表示当前没有指向任何有效对象。

## 📌 这节只解决什么

- 指针变量里到底存什么
- 什么情况下能解引用 `*p`
- `NULL` 和 `'\0'` 为什么不能混

## ✅ 最小正确写法

```c
int x = 10;
int *p = &x;

if (p != NULL) {
    printf("%d\n", *p);
}
```

## 🔁 代码桥

- `int *p`
  - `p` 是指针变量
  - 里面存的是某个 `int` 的地址
- `*p`
  - 不是地址
  - 是“到这个地址里把值取出来”
- `NULL`
  - 表示“现在没指向有效对象”
  - 不能直接 `*NULL`

## ⚠️ 易混点

| A | B | 怎么区分 |
|---|---|---|
| `p` | `*p` | `p` 是地址；`*p` 是地址里的值 |
| `NULL` | `'\0'` | 前者给指针用；后者给字符/字符串用 |
| `&x` | `x` | 前者是地址；后者是值 |

## 💥 高频坑

- 指针没初始化就解引用
- 没判 `NULL` 就直接 `*p`
- 把 `NULL` 和 `0`、`'\0'` 随意混用
