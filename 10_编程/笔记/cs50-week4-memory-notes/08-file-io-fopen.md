# 8）fopen 模式：r、w、a、b、+

## 🎯 一句话结论

> `r` 要求文件先存在；`w` 会清空重写；`a` 会保留旧内容并追加到末尾。

## 📌 这节只解决什么

- 看到题目时，先该选哪种打开模式
- `r / w / a`、`+`、`b` 各自是什么意思
- 为什么有些模式“能用”，但并不合适

## ✅ 最小正确写法

```c
FILE *fp = fopen("data.txt", "r");
if (fp == NULL) return 1;

/* 读或写 */

fclose(fp);
```

## 🔁 代码桥

### 先看主动作

- `r` = 读
- `w` = 重写
- `a` = 追加

### 再看补充动作

- `+` = 在原动作基础上再加另一种权限
- `b` = 用二进制方式打开

### 最后看常见组合

- `rb` = 读已有二进制文件
- `rb+` = 读写已有二进制文件
- `wb+` = 清空后再读写
- `ab+` = 追加并允许读

## ⚠️ 易混点

### `"w"` vs `"a"`

- `"w"`：旧内容会被清空
- `"a"`：旧内容保留，新内容接后面
- 一句话区分：想续写用 `a`，想重建用 `w`

### `"rb"` vs `"rb+"`

- `"rb"`：只读
- `"rb+"`：读写
- 一句话区分：有没有 `+`，决定能不能再写

## 💥 高频坑

- `if (fp = NULL)` 写成赋值
- 明明想追加，却写成 `"w"`
- 明明只想读已有文件，却用了会清空文件的模式
