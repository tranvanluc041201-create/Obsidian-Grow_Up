# 【新手向】二级指针 ** 详解

> 一句话：`*`是开门钥匙，`*p`开一次门，`**pp`开两次门。

---

# 🏠 类比理解：钥匙与房间

想象一个宾馆：

```
房间201：住着整数 10（变量x）

前台有个箱子p：里面放着钥匙 "201"（指向x的指针）

经理室有个箱子pp：里面放着钥匙 "前台箱子p的位置"（指向p的指针）
```

### 开门过程

| 操作 | 结果 | 解释 |
|------|------|------|
| `x` | 10 | 直接进201房间 |
| `*p` | 10 | 用p里的钥匙(201)开门 → 看到10 |
| `**pp` | 10 | 先用pp找到p，再用p找到x → 看到10 |

---

## 📊 代码对应

```c
int x = 10;        // 房间201，住着10
int *p = &x;       // 箱子p，存着钥匙"201"
int **pp = &p;     // 箱子pp，存着钥匙"箱子p的位置"
总结:一次*就取指向的变量的内容
```

**内存图**：
```
地址从高到低：

┌─────────────────┐ 0x1000
│ pp (int**)      │  里面存着：0x500
│ 经理室的箱子    │
└────────┬────────┘
         │
         ▼
┌─────────────────┐ 0x500
│ p (int*)        │  里面存着：0x201
│ 前台的箱子      │
└────────┬────────┘
         │
         ▼
┌─────────────────┐ 0x201
│ x (int)         │  里面存着：10
│ 201房间         │
└─────────────────┘
```

---

## 🔍 一步步拆解




### 第一步：`*pp` 是什么？

```c
*pp  // 对pp解引用一次
```

过程：
1. 打开经理室的箱子pp
2. 看到里面写着 "0x500"
3. 去0x500位置，找到前台的箱子p

结果：`*pp` = **p的值** = `&x`（x的地址）

---

### 第二步：`**pp` 是什么？

```c
**pp  // 对pp解引用两次
```

过程：
1. 第一次 `*pp` → 找到箱子p，得到 `&x`
2. 第二次 `*&x` → 去x的地址，得到 **10**

结果：`**pp` = **x的值** = `10`

---

## 📝 所有表达式对照

```c
int x = 10;
int *p = &x;
int **pp = &p;
```

| 表达式        | 结果      | 解释              |
| ---------- | ------- | --------------- |
| `x`        | 10      | 直接访问x           |
| `*p`       | 10      | p指向x，*p就是x      |
| `**pp`     | 10      | pp→p→x，两次解引用得到x |
| `*pp`      | &x（p的值） | 只开一次门，得到p       |
| `pp[0]`    | &x      | 等价于 `*pp`       |
| `pp[0][0]` | 10      | 等价于 `**pp`      |

---



## 🔥 为什么需要二级指针？

### 场景1：动态二维数组（指针数组）

```c
// 想象一个酒店，每层楼的房间数不同
int **hotel;  // hotel是经理室的箱子

hotel = malloc(3 * sizeof(int*));  // 3层楼
// hotel[0] = 1楼的所有房间的钥匙
// hotel[1] = 2楼的所有房间的钥匙
// hotel[2] = 3楼的所有房间的钥匙

hotel[0] = malloc(4 * sizeof(int));  // 1楼有4间房
hotel[1] = malloc(3 * sizeof(int));  // 2楼有3间房
hotel[2] = malloc(5 * sizeof(int));  // 3楼有5间房

// 访问2楼第3间房：
hotel[1][2] = 100;  // 先找到2楼，再找第3间房
```

---

### 场景2：函数修改指针本身

```c
void allocate(int **pp, int size) {
    *pp = malloc(size * sizeof(int));  // 修改外面的指针
}

int main() {
    int *p = NULL;           // 我想让p指向新内存
    allocate(&p, 10);        // 传p的地址给函数
    // 现在p指向分配的内存了！
    free(p);
    return 0;
}
```

**为什么不能只用一级指针？**
```c
void wrong(int *p) {
    p = malloc(10);  // 只修改了函数内部的p，外面的p还是NULL
}
```

## 📝 练习题

### 练习1：追踪
```c
int a = 1, b = 2;
int *p1 = &a, *p2 = &b;
int *arr[2] = {p1, p2};
int **pp = arr;

// *pp = ?
// **pp = ?
// *(pp+1) = ?
// **(pp+1) = ?
```

**答案**：
- `*pp` = &a（p1的值）
- `**pp` = 1（a的值）
- `*(pp+1)` = &b（p2的值）
- `**(pp+1)` = 2（b的值）

### ASCII图
变量      地址       值
a            0x100    1  
b            0x200    2
arr          0x300   0x100
pp          0x400   0x300

---

### 练习2：动态数组
```c
int **m = malloc(2 * sizeof(int*));
m[0] = malloc(3 * sizeof(int));
m[1] = malloc(3 * sizeof(int));

m[0][0] = 1; m[0][1] = 2; m[0][2] = 3;
m[1][0] = 4; m[1][1] = 5; m[1][2] = 6;

// m[1][2] = ?
// *(*(m+1)+2) = ?
```

**答案**：都是6

---

## 💡 记忆口诀

> **一层 * 得地址，两层 ** 得值**  
> **pp[i][j] 就是 *(*(pp+i)+j)**  
> **修改指针用二级，动态数组要用它**

---

## ⚠️ 常见错误

| 错误写法 | 问题 |
|----------|------|
| `int **pp; **pp = 10;` | pp未初始化，野指针崩溃 |
| `**pp = *p` | 这是比较值，不是类型说明 |
| 混淆 `*pp` 和 `**pp` | 前者是地址，后者是值 |
