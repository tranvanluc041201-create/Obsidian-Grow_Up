# Q6 指针定义语法视力测试

## 🎯 题目 / 场景

指针定义语法辨析，哪个正确？

```c
A. char a='A' b='B';           // ❌ 缺少逗号
B. float *a = b = 10.0;        // ❌ b未定义
C. int a=10, *b=&a;            // ✅ 正确
D. float *a, b=&a;             // ❌ b是float不是指针
```

---

## 💥 发生了什么

这种题考的不是逻辑，是**眼睛和基本语法规则**。

---

## 🔍 根因分析

| 选项 | 错误原因 |
|------|---------|
| **A** | `a='A'` 和 `b='B'` 中间缺了逗号 |
| **B** | `float *a` 初始化时 `b` 还未定义 |
| **C** | ✅ `int a=10` 定义变量；`*b=&a` 定义指针并指向a |
| **D** | `float *a, b=&a` 中 `b` 被定义为float（非指针） |

**关键语法**：
```c
int *a, b;   // a是指针，b是int
int *a, *b;  // a和b都是指针
```

## ✅ 正确规则与修正

```c
// ✅ 正确写法
int a = 10;
int *b = &a;  // 指针定义并初始化

// ✅ 一行定义多个
int x = 1, y = 2, *p = &x;
```
