# 2.6 返回局部变量地址

## 🎯 题目 / 场景

返回局部变量地址导致野指针：

```c
int* f() {
    int a = 10;  // 局部变量，在栈上
    return &a;   // ❌ 返回局部变量的地址
}

int main() {
    int *p = f();  // p指向已释放的内存
    printf("%d\n", *p);  // 💥 未定义行为！
    return 0;
}
```

---

## 💥 发生了什么

- **现象**：可能输出10，也可能输出乱码，甚至崩溃
- **原因**：函数返回后，局部变量的栈帧被销毁，地址指向"垃圾数据"

---

## 🤔 我当时怎么想的

> "函数返回指针很常见，应该没问题..."
> "既然能编译通过，就应该能运行..."
> **盲区**：不知道局部变量的生命周期只限于函数内部，返回后内存被回收。

---

## 🔍 根因分析

### 表面原因
> 返回了局部变量的地址

### 深层原理

**栈帧的生命周期**：
```
调用f()时：
┌─────────────────────┐
│  局部变量 a = 10    │  ← f()的栈帧
│  (地址: 0x7ffd100)  │
└─────────────────────┘
返回&a (0x7ffd100)

f()返回后：
┌─────────────────────┐
│  原a的位置          │  ← 栈帧已释放，内容不确定
│  (可能被其他数据覆盖) │
└─────────────────────┘
p仍指向0x7ffd100，但数据已无效！
```

**为什么有时还能正确输出？**
- 栈帧释放后，内存内容**暂时**没被覆盖
- 但这是**未定义行为**，随时可能出错
- 一旦有其他函数调用，该位置会被新数据覆盖

### 识别信号
> **返回指针的3种安全方式**：

> 1. 返回静态变量（`static`）
> 2. 返回动态分配的内存（`malloc`）
> 3. 通过参数传入缓冲区（由调用者管理）

---

## ✅ 正确规则与修正

```c
// ✅ 方法1：静态变量（生命周期贯穿程序）
int* f() {
    static int a = 10;  // 静态变量
    return &a;          // 安全，但所有调用共享同一个a
}

// ✅ 方法2：动态分配（调用者负责free）
int* f() {
    int *a = malloc(sizeof(int));
    if (a != NULL) {
        *a = 10;
    }
    return a;  // 调用者必须free！
}

// ✅ 方法3：传入缓冲区（最常用）
void f(int *result) {  // 由调用者提供缓冲区
    *result = 10;
}

int main() {
    int value;
    f(&value);  // 传入局部变量的地址
    printf("%d\n", value);  // 安全！
    return 0;
}
```

## 🔁 一题多变 / 横向关联（视情况补充）

### 同类变式
- **更隐蔽的野指针**
  ```c
  int* get_array() {
      int arr[10] = {1,2,3};
      return arr;  // 同样错误！
  }
  ```

- **字符串版本**
  ```c
  char* get_string() {
      char str[] = "Hello";  // 局部数组
      return str;            // 错误！
  }
  
  // ✅ 正确：字符串常量
  char* get_string() {
      char *str = "Hello";   // 字符串常量在全局区
      return str;            // 安全
  }
  ```

### 工程启示
> **返回指针的黄金法则**：
> - 绝不返回局部变量的地址
> - 如果要返回指针，确保指向：
>   - 静态变量（注意线程安全）
>   - 动态分配的内存（调用者free）
>   - 调用者传入的缓冲区
>
> **调试技巧**：使用AddressSanitizer (`-fsanitize=address`) 自动检测野指针。

| 场景 | 解决方案 |
|------|---------|
| **返回局部变量地址** | ❌ 绝对禁止 |
| **需要返回数据** | 用static / malloc / 传入参数 |
| **临时数据** | 传入缓冲区（最常用） |
| **持久数据** | malloc（记得free） |
| **共享数据** | static（注意线程安全） |
