# 4.5 指针地址打印格式

## 🎯 题目 / 场景

```c
int x = 10;
int *p = &x;

printf("%d\n", p);   /* ❌ */
printf("%p\n", p);   /* ✅ */
```

---

## 💥 发生了什么

- 预期：打印指针地址
- 实际：用 `%d` 打印指针，格式不匹配
- 结果：输出错误、警告，甚至未定义行为

---

## 🤔 我当时怎么想的

> “地址本质上也是数字，那用 `%d` 打印应该差不多。”

---

## 🔍 根因分析

### 表面原因
- `printf` 的格式说明符和参数类型不匹配

### 深层原因
- 指针类型和 `int` 类型不是一回事
- `%p` 专门用于打印地址
- 通常应把参数转成 `(void *)`

### 识别信号
- 看到“打印地址”这类需求时，先想 `%p`

---

## ✅ 正确规则与修正

### 正确规则
- 打印地址：`%p`
- 传参：`(void *)指针`

### 最小修正代码
```c
int x = 10;
int *p = &x;
printf("%p\n", (void *)p);
```
