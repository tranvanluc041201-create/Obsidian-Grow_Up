# 编程错题本

## 主目录

> 总索引：[[10_编程/元信息/编程总索引|编程总索引]]

- `01_Compile_Link`：语法、声明、宏、编译期陷阱
- `02_Memory_Address`：指针、数组、传参、地址、内存
- `03_Logic_Control`：分支、循环、短路、执行顺序
- `04_IO_Buffer`：输入输出、格式串、缓冲区、文件基础
- `05_Data_Representation`：整数、浮点、字符、进制、类型表示

## 使用规则

1. 新错题只记到这 5 个主分类
2. 文件名统一用：`[模块编号].[序号] 简短描述.md`
3. 正文优先写：错误直觉、根因、最小修正、自检
4. 遇到复杂卡题，再改用 `99_Templates/编程/深度复盘模板`

## 维护原则

- 主目录是当前版本
- 以后不再恢复 `错误/错题本-old` 这类旧命名
- 同类错题累计到 2 次以上，建议补一篇学习笔记
