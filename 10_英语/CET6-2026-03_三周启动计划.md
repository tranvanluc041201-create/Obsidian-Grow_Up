# CET6 三周启动计划（2026-03）

> [!summary]
> 当前默认版本：
> - `2026年6月`：首战诊断 + 尽量冲高
> - `2026年12月`：主目标 `550`
> - 主攻：听力
> - 次主攻：写作 / 翻译
> - 阅读：维持手感
> - 单词：不单练，寄生在真题里

## 先看执行原则

1. 每晚只跑 `60 分钟` 主流程。
2. 主流程顺序固定：听力 → 表达/翻译 → 真题词块。
3. `技术阅读 / AI 泛听` 只能做副轨，不能挤占主轨。
4. 每次听力只抓 `3` 个盲点，不做大而全复盘。
5. 连续性比强度更重要，状态差时改跑 `MVP 20 分钟版`。

## 今晚默认骨架

### 1. 听力真题 `35-40 分钟`

- `5 分钟` 盲听：不暂停，估算理解度
- `20-25 分钟` 精听：句子循环 + 打字验证
- `10 分钟` 核对：只记 `3` 个盲点

优先记录的错误类型：

1. `Linking`：连读、弱读、失爆
2. `Speed`：语速快，能听到但来不及切分
3. `Vocab`：词本身不认识
4. `Comprehension`：词听到了，但意思没接上

### 2. 写作 / 翻译 `10-15 分钟`

- 只做一件小事：
  - 整理 `1` 张表达卡
  - 或改写 `1-2` 个句子
  - 或摘 `1` 个可复用句型

### 3. 真题词块 `5-10 分钟`

- 只保留今天材料里最值钱的 `5-8` 个词块
- 优先保留：
  - 听力反复听不清的词
  - 阅读选项常见替换词
  - 写作/翻译能直接套用的表达

## 3 周安排

### 第 1 周：把流程跑顺

**目标**：先形成连续性，不追求做很多。

- 周一：听力主攻夜
- 周二：听力轻量 + 阅读维持
- 周三：听力主攻夜
- 周四：听力轻量 + 写作/翻译
- 周五：听力主攻夜
- 周六：小整合
- 周日：周复盘或休整

**默认晚间安排**

- 周一 / 周三 / 周五
  - `40 分钟` 听力
  - `15 分钟` 表达/翻译
  - `5 分钟` 真题词块
- 周二 / 周四
  - `25 分钟` 听力盲点回放
  - `25 分钟` 阅读真题
  - `10 分钟` 词块 / 表达
- 周六
  - `30 分钟` 听力检验
  - `20 分钟` 写作或翻译
  - `10 分钟` 整理卡片
- 周日
  - 用 [[99_Templates/CET6/T7_周复盘|周复盘模板]] 做一次短复盘
  - 如果状态差，直接休整

**第 1 周验收**

1. 至少留下 `6` 次训练记录
2. 至少记下 `6` 个听力盲点
3. 能说出自己最常见的 `2` 类听力错误

### 第 2 周：开始稳定输出

**目标**：不换流程，只提高质量。

- 听力精听必须打字验证
- 阅读训练先计时，不查词
- 写作/翻译开始固定积累表达

**第 2 周验收**

1. 听力错因不再只写“没听懂”
2. 阅读错题至少写出 `2` 条“下次规则”
3. 写作/翻译至少积累 `4-5` 条可复用表达

### 第 3 周：做第一次小诊断

**目标**：看闭环是否形成，而不是盲目加量。

- 周一 / 周三：听力主攻
- 周二 / 周四：阅读维持 + 听力回放
- 周五：写作 / 翻译集中一次
- 周六：做一次缩短版模拟
- 周日：认真复盘

**周六建议**

1. `25-30 分钟` 听力集中训练
2. `20-25 分钟` 阅读计时
3. `10 分钟` 写一句总结

复盘只回答 3 个问题：

1. 我最常错什么？
2. 我最有效的方法是什么？
3. 下周只改哪一条？

**第 3 周验收**

1. 能明确说出最主要的 `3` 类听力错因
2. 不再把“背单词”当主任务
3. 晚间 `60 分钟` 基本稳定

## MVP 20 分钟版

当晚状态差时，不许直接断更，改跑最低版本：

1. `15 分钟` 听力盲点回放
2. `5 分钟` 记录 `1` 个错点

做到这一步，当天就算不断更。

## 材料规则

1. 先准备最近 `4` 套六级真题
2. 第一阶段只用这 `4` 套，不到处换资料
3. 材料必须同时有：音频、原文、题目、答案解析

## 副轨规则：技术阅读 / AI 泛听

> [!warning]
> 这部分的作用是保持续航，不是替代六级训练。

允许保留，但必须满足：

1. 每周 `2-3` 次
2. 每次 `10-15` 分钟
3. 主轨做完后才能做
4. 当天主轨没做完，副轨直接取消
