```
### 3. 控制链路讲解（PWM / 闭环 / 开环）

#### PWM 的本质
**脉宽调制**：用"开关开多久"来控制平均能量 [1]

类比：水龙头"开-关-开-关"，你控制"每次开着的时间比例"，就控制平均流量

- **Comparator vs ramp** 本质上是在做"定时"：
  - ramp 是一个每周期从 0→1 的"计时尺"
  - Limiter 输出是你想要的"时间比例"（即占空比 $d$）
  - 比较器说：`当 ramp < d 时，PWM=1；否则 PWM=0`

- 只要 **ramp 与 d 同量纲**（0..1 对 0..1），PWM 的占空比就自然等于 $d$

#### 开环 vs 闭环
- **开环**：固定给一个 $d$（比如 0.3），不管输出漂到哪。像"按经验每天吃同样剂量的药"
- **闭环**：测 Vfb，和 Vref 比，算出误差 $e$，再改 $d$。像"测血糖→算差→调胰岛素剂量"

#### 控制链路的脆弱点
你的链路：`e → PI → Limiter(0..1) → Comparator vs ramp(0..1@50k) → PWM → Switch` [1]

| 模块 | 问题 | 优先级 |
|---|---|---|
| **PI** | 可能饱和、积分 wind-up | ⭐⭐ |
| **Limiter** | 量纲错（范围不是 0..1） | ⭐⭐⭐⭐⭐ |
| **Comparator** | 极性反 | ⭐⭐⭐⭐ |
| **MOSFET** | 浮地导致 Vgs 不成立 | ⭐⭐⭐⭐⭐ |

## ⚠️ 致命坑表（现象 → 根因 → 验证 → 修复）

| 坑 | 典型现象 | 根因 | 最快验证（10秒） | 修复方案 |
|---|---|---|---|---|
| **参考点不一致** | 输出 -600k、测量不对、80≠100 | 你以为同地，实际不同节点 [1] | 传感器负端拉到 VDC− 黑点 [1] | 统一参考点 [1] |
| **MOSFET 浮地** | PWM 在跳，支路不开 | Gate 对地，Source 漂移，Vgs 实际很小 [1] | 测 Vgs=V(gate)-V(source) [1] | 改 Ideal/Controlled [1] |
| **Limiter 顶死** | VC1/2/3 全 0 或全 1 | 误差极性错/量纲不匹配 [1] | 同屏看 e、PIout、Limout、ramp [1] | 校验极性；确认 Lim=0..1 [1] |
| **回流路径错** | 输出起不来、振荡、异常波形 | 体二极管方向错/回流不通 [1] | 断开支路逐个验证 [1] | 按论文核对方向 [1] |

## 🎯 排错决策树（优先级固定）

```

[Start] 发现输出不对 / 数值离谱  
│  
├─→ (A) 是否出现离谱数值？(巨负/巨正/爆炸，如 -600k)  
│ │  
│ ├─→ A1: 同屏查看 Limout 是否卡 0/1？  
│ │ ├─ 是 → 修 "误差极性/量纲匹配"(e 方向, Lim=0..1, ramp=0..1)  
│ │ └─ 否 → A2  
│ │  
│ └─→ A2: 统一参考点验证（传感器负端硬拉到 VDC−）  
│ ├─ 立刻正常 → 根因 = 参考点不一致（END）  
│ └─ 仍异常 → A3  
│  
└─→ (B) 数值不离谱但偏差大（达不到目标/80≠100）  
│  
├─→ B1: Vfb 是否取自 Cout 两端？  
│ ├─ 否 → 修测量点（END）  
│ └─ 是 → B2  
│  
├─→ B2: MOSFET/开关是否真导通？  
│ ├─ 不导通 → 修 Vgs/器件模型（END）  
│ └─ 导通 → B3  
│  
└─→ B3: 再看 PI（饱和/积分风up/参数过激）  
└─ 先降 Kp/Ki 或加 anti-windup
