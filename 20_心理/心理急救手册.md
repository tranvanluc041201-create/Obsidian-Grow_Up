## 🔄 事后反刍急救卡

**症状**：反复回想社交场景，分析自己哪里做得不对，想象别人的评价

### 🎯 5-4-3-2-1 接地技术（立即执行）

深呼吸一次，然后：

| 步骤 | 行动 | 示例 |
|------|------|------|
| 5 | 说出5样能看到的东西 | "手机、台灯、书、水杯、窗帘" |
| 4 | 说出4样能触摸到的东西 | "桌子的木纹、手机的温度、衣服的质地..." |
| 3 | 说出3样能听到的声音 | "空调声、外面车流、自己的呼吸..." |
| 2 | 说出2样能闻到的气味 | "咖啡、洗发水的味道..." |
| 1 | 说出1样能尝到的味道 | "刚才喝的水、嘴里残留的食物..." |

> 🕐 **耗时**：1-2分钟  
> ✅ **目标**：把注意力从"脑子里"拉回"当下"

---

### 🎯 反刍中断三问

如果接地后还在想，问自己：

```
1. 【事实检验】这件事，实际有人当面批评我吗？
   → 没有 → "这是我的想象，不是事实"
   → 有 → 进入问题2

2. 【影响检验】这件事一年后还重要吗？
   → 不会 → "大脑在保护我，但我现在安全了"
   → 会 → 进入问题3

3. 【行动检验】现在我能做什么来改变吗？
   → 能 → 写下来，设定时间去做
   → 不能 → 接受"暂时无法解决"，转移注意力
```

---

### 🎯 注意力转移清单

反刍时，做这些（选一个）：

- [ ] 背诵10个C语言关键字
- [ ] 做5道选择题
- [ ] 去洗手/喝水/开窗（物理移动）
- [ ] 听一首快节奏的歌，跟着节奏拍手
- [ ] 给某个物品详细描述（比如"这支笔是黑色的，按动式，笔身有..."）

**禁忌**：刷短视频、刷社交媒体（会加重比较和焦虑）

## 👁️ 被监视感急救卡

**症状**：和哥哥同处一室时感觉被盯着、评判，动作僵硬，不敢自然做事

### 🎯 认知重构

**第一步：区分"事实"和"感受"**

| 感受（主观） | 事实（客观） |
|-------------|-------------|
| "他在盯着我看" | "他在我视线范围内，但我不知道他在看什么" |
| "他觉得我恶心" | "他没有说话，这是我的推断" |
| "他在等我出错" | "我不知道他的想法" |

> 💡 **记住**：你不是他肚子里的蛔虫，你的想法≠他的想法

---

### 🎯 物理隔离策略

**立即能做的：**

| 场景 | 应对 |
|------|------|
| 吃饭时 | 戴耳机听音乐/播客；背对他坐；快速吃完离开 |
| 同处一室 | 创造"视觉屏障"（书本、电脑挡住视线）；专注于手头的事 |
| 必须经过时 | 目不斜视，快步通过，不停留 |
| 他说难听话时 | 不回应、不对视、不解释（灰色岩石法） |

---

### 🎯 心理盾牌技术

想象你周围有一个**透明保护罩**：
- 他的眼神、话语碰到保护罩会滑开
- 保护罩内是你的空间，他是外人
- 默念："这是他的问题，不是我的"

---

### 🎯 权力回收

他让你不舒服，但**你可以选择**：
- 选择不看他
- 选择不听他说话
- 选择不把他的评价当真

> 🔑 **关键认知**：他对你的态度反映的是他自己，不是你。

## ⏰ 事前焦虑急救卡

**症状**：考试前、重要事件前反复担心"要是失败了怎么办"

### 🎯 焦虑外化技术

**写下你的担心**：

```
我担心：_________________________
最坏的结果：_____________________
如果发生了，我会：______________
_________________________________
```

> 💡 **原理**：焦虑在脑子里是模糊的怪兽，写出来就具体了，可管理了

---

### 🎯 概率检验

问自己：

```
这件事发生的概率是多少？（0-100%）
___%

过去类似的担心，实际发生的比例是多少？
___%

即使发生了，我真的无法承受吗？
是的 / 不是（圈选）
```

---

### 🎯 准备vs担忧

区分"有效准备"和"无效担忧"：

| 有效准备 | 无效担忧 |
|---------|---------|
| 现在能做的事 | 未来可能发生的事 |
| 可以行动的 | 无法控制的 |
| 做完就放下 | 反复想停不下 |

**原则**：如果这件事现在做不了，就把它从脑子里拿出来，放到"未来待办清单"

## 🏠 家庭压力急救卡

**症状**：和家人相处后情绪低落、自我怀疑、愤怒或无力

### 🎯 情绪急救流程

**Step 1：离开现场**
- 找个理由离开（上厕所、拿东西、回房间）
- 物理距离 → 情绪降温

**Step 2：情绪命名**
- 我现在感到：______（愤怒？委屈？羞耻？无力？）
- 命名情绪本身就能降低强度

**Step 3：自我关怀对话**

```
我知道这很难受。
这不是我的错。
他们的态度反映的是他们自己。
我现在安全了。
我会照顾好自己的。
```

---

### 🎯 哥哥行为解读

他的嫌弃、敌意，可能来源：
- 他自己也有未解决的青春期困惑
- 他对亲密关系的扭曲理解
- 他把自己的不安全感投射到你身上

> 🔑 **这不是你的错，也不是你的责任去修复他**

---

### 🎯 边界建立

**你可以设定的边界**：
- 不和他单独相处
- 不回应他的挑衅
- 不接受他的情绪垃圾
- 在心里把他"降级"（不是重要的评价者）

## 🧠 认知-情绪失调急救卡

**症状**："我知道不该在意，但就是做不到"

### 🎯 理解认知vs情绪

```
认知（理性层）：知道道理
        ↓
情绪（感性层）：身体记忆、条件反射
        ↓
行为（行动层）：实际反应
```

**三者不同步是正常的**，需要时间让情绪层"跟上"

---

### 🎯 允许情绪存在

不要说："我不应该难过"
要说："我现在很难过，这很正常，我可以难过"

**情绪像海浪**：
- 来了 → 承认它（"我看到你了"）
- 峰值 → 坚持住（会过去的）
- 退去 → 恢复正常

---

### 🎯 小步验证法

不要期望"一夜之间不在意"
而是：

```
之前在意程度：90分
目标：在意程度降到70分（不是0分）

每次做到一点点就肯定自己：
"虽然还在意，但我比上次少反刍了10分钟"
```

## 💥 情绪崩溃急救卡

**症状**：突然想哭、心跳加速、呼吸困难、感觉失控

### 🎯 紧急稳定技术（TIPP）

| 技术 | 做法 | 原理 |
|------|------|------|
| **T**emperature | 用冷水洗脸或握冰块 | 激活潜水反射，降低心率 |
| **I**ntense exercise | 原地快速跑30秒或俯卧撑 | 释放肾上腺素 |
| **P**aced breathing | 吸气4秒→憋气4秒→呼气6秒 | 激活副交感神经 |
| **P**rogressive relaxation | 用力握拳5秒→突然松开，感受放松 | 对比学习放松感 |

**选一个做，直到感觉平稳**

---

### 🎯 安全确认

对自己说：

```
我现在是安全的。
这种感觉会过去的。
我以前挺过来了，这次也可以。
这只是身体的一个反应，不是危险信号。
```

---

### 🎯 崩溃后恢复

1. **不评判自己**：崩溃不是软弱，是身体在释放压力
2. **简单自我照顾**：喝水、吃点东西、躺下休息
3. **不立即处理复杂事务**：等状态恢复再说
4. **记录触发点**：帮助以后预防

## 💬 最后的提醒

**你不是一个人。**

你描述的这些感受——敏感、反刍、高警觉——都是人在长期压力下形成的保护机制。它们曾经帮助过你，现在可能过度了，但这不意味着你有问题。

**改变需要时间。**
不要期待读完这个手册就能解决所有问题。把它当成一个工具箱，每次拿一个工具试试，找到最适合你的。

**你已经很勇敢了。**
愿意面对这些、寻求帮助、尝试改变——这本身就需要巨大的勇气。
