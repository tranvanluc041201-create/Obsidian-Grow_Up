# 戒色重建系统

> [!summary]
> 这套系统只做三件事：稳住晚上、切断色情、把精神面貌拉回来。

## 先从这里用

- 今天先打开 [[20_心理/戒色重建系统/01_14天重置计划|14天重置计划]]
- 晚上具体照着 [[20_心理/戒色重建系统/04_具体执行流程|具体执行流程]] 走
- 晚上冲动上来时只看 [[20_心理/戒色重建系统/02_冲动急救卡|冲动急救卡]]
- 每天只维护一页当天日记录；第一天先用 [[20_心理/戒色重建系统/每日记录/2026-03-09|今天的日记录]]
- 后续新建日记录时，复制 [[20_心理/戒色重建系统/03_每日记录模板|每日记录模板]]
- 每周末写一页 [[20_心理/戒色重建系统/每周复盘/2026-W11|每周复盘]]

## 自动化入口

- `QuickAdd` 命令已添加：`打开今日戒色日记录`
- 如果你直接在 `20_心理/戒色重建系统/每日记录` 新建一个 `.md` 文件，`Templater` 会自动套用戒色日记录模板

## 你的硬规则

- 色情内容清零
- 手机不进床
- 06:45 固定起床
- 晚上先降刺激，再谈入睡
- 失控后不自罚，只复盘触发链

## 近14天执行概览

```dataviewjs
const pages = dv.pages('"20_心理/戒色重建系统/每日记录"')
  .where(p => p.type === "戒色日记录" && p.date)
  .sort(p => p.date, "desc")
  .array();

const labels = [
  { text: "不看色情内容", label: "无色情" },
  { text: "不自慰", label: "无自慰" },
  { text: "手机不进床", label: "手机离床" },
  { text: "23:15 开始睡前降刺激", label: "睡前降刺激" },
  { text: "06:45 起床", label: "固定起床" },
  { text: "20分钟运动或快走", label: "活动达标" }
];

function taskDone(page, text) {
  const tasks = page.file.tasks ?? [];
  return tasks.some(task => task.text.trim() === text && task.completed);
}

function streak(text) {
  let count = 0;
  const ordered = [...pages].sort((a, b) => String(b.date).localeCompare(String(a.date)));
  for (const page of ordered) {
    if (taskDone(page, text)) {
      count += 1;
      continue;
    }
    break;
  }
  return count;
}

const recent = pages.slice(0, 14);
dv.paragraph(`已记录天数：${recent.length} 天`);
dv.table(
  ["指标", "完成天数"],
  labels.map(item => [
    item.label,
    recent.filter(page => taskDone(page, item.text)).length
  ])
);
dv.paragraph(`当前连续无色情：${streak("不看色情内容")} 天`);
dv.paragraph(`当前连续无自慰：${streak("不自慰")} 天`);
```

## 使用顺序

1. 白天只做计划页里的启动任务。
2. 晚上只维护当天的日记录，不做长篇自责。
3. 如果破了，不解释，直接回到计划页和急救卡。
