## 使用方法

1. 打开对应模板
2. 复制内容
3. 粘贴到 `10_编程/笔记/` 或 `10_编程/错题本/` 目录
4. 按模板格式填写

补充：

- `思维导图协作模板` 是协作流程卡，不是普通笔记模板
- 用户说 `思维导图` / `脑图` 时，先用它判断图型、层级和验收标准
- 如果用户连知识点范围都说不清，AI 再按需补充候选骨架或拆图建议
