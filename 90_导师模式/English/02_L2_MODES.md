## 听力训练模式

### Triple Listen 详细流程

**Step 1: Blind (5min)**
- No pause, no text
- Estimate: I understood ___%

**Step 2: Intensive (30-40min)**
- Sentence loop 3x
- Type what you hear
- Mark: [time] [what I heard] [actual]

**Step 3: Check (5min)**
- Compare with transcript
- Categorize mistake type
- Save to `10_英语/听力/错题/`

### 错误类型

| Type | Fix |
|------|-----|
| Linking | Shadow the phrase 5x |
| Speed | 0.8x → 1.0x → 1.2x |
| Vocab | Add to word list |
| Comprehension | Focus on key points only |

## 写作训练模式

### PARROT 格式

| Field | Fill |
|-------|------|
| Phrase | Original sentence |
| Use | When to use it |
| Replace | Alternative words |
| Source | Where found |
| My sentence | Your imitation |

### 默认流程

1. 先指出原句最核心的问题
2. 让用户自己改一次
3. 再批改用户版本
4. 最后给更自然的参考版本
5. 只抽 `1-2` 个最值得复用的表达

---

## 翻译训练模式

### 默认流程

1. 先判断中文句子的核心关系：主干、修饰、逻辑
2. 再逐句指出最关键的问题
3. 先让用户自己改一次
4. 再给逐句修改建议
5. 最后给更优版本
6. 抽 `1-2` 个可复用表达
