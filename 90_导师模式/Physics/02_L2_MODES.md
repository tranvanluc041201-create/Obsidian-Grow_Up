## 1. 力学分析模式

### 适用场景
质点运动、连接体、斜面、弹簧系统、圆周运动

### 专用协议

#### Step 1: 系统选择决策树
```
是否求内力？
├─ 是 → 隔离体法
│       └─ 画每个物体的FBD
└─ 否 → 整体法
        └─ 画系统整体的FBD
        └─ 只画外力，内力抵消
```

#### Step 2: FBD绘制清单
- [ ] 正确隔离研究对象（画虚线框）
- [ ] 标出所有接触力（弹力、摩擦力）
- [ ] 标出所有场力（重力、电场力）
- [ ] 建立坐标系（尽量让加速度沿轴）
- [ ] 标注角度和几何关系

#### Step 3: 方程建立顺序
1. 对每个隔离体写 ΣF = ma
2. 约束方程（如绳子不可伸长：a₁ = a₂）
3. 摩擦关系（f ≤ μN，判断是否滑动）
4. 检查：方程数 = 未知数个数

### 常见Pattern
| Pattern | 说明 |
|---------|------|
| `FBD-连接体` | 先整体求加速度，后隔离求内力 |
| `FBD-临界问题` | 找临界条件（f_max, N=0, T=0） |
| `FBD-变力问题` | 建立微分方程或能量关系 |

### 高频Error_Type
- **E1-画图**: 漏掉法向力、摩擦力方向错
- **E2-选系统**: 该隔离时用了整体，求内力时没隔离
- **E4-选原则**: 复杂变力问题硬用牛顿第二定律

## 2. 能量动量模式

### 适用场景
碰撞、爆炸、弹簧系统、曲线运动、多过程问题

### 守恒条件检查清单

#### 机械能守恒
- [ ] 只有保守力做功？
- [ ] 无摩擦、无非弹性碰撞
- [ ] 无有动力/阻力做功

#### 动量守恒
- [ ] 系统合外力 = 0？
- [ ] 或内力 ≫ 外力（碰撞瞬间）
- [ ] 明确在哪个方向守恒

#### 角动量守恒
- [ ] 合外力矩 = 0？
- [ ] 或有心力场
- [ ] 转轴明确且固定

### 决策流程图
```
问题类型？
├─ 碰撞/爆炸 → 动量守恒 + 能量关系（弹性/非弹性）
├─ 曲线运动 → 能量守恒（找等势点）
├─ 转动问题 → 角动量守恒
└─ 多过程 → 分段分析，找连接点
```

### 常见Pattern
| Pattern | 说明 |
|---------|------|
| `能量-势能曲线` | 利用E_p-x图分析运动范围 |
| `动量-质心系` | 碰撞问题转到质心参考系 |
| `能量-有效势` | 有心力场用有效势分析 |

## 3. 刚体转动模式

### 专用协议

#### 转轴选择原则
1. 尽量让未知力通过转轴（力矩为0）
2. 尽量利用对称性
3. 固定轴 vs 瞬轴的选择

#### 转动惯量常用值
| 物体 | 转轴 | I |
|------|------|---|
| 细杆 | 中心 ⊥ | (1/12)mL² |
| 细杆 | 端点 ⊥ | (1/3)mL² |
| 圆盘 | 中心 ⊥ | (1/2)mR² |
| 球体 | 直径 | (2/5)mR² |

#### 纯滚动条件
- 运动学：v = Rω, a = Rα
- 摩擦：静摩擦，不做功（接触点瞬时静止）
- 能量：可用机械能守恒

## 4. 振动波动模式

### 简谐振动

#### 判断标准
- 回复力 F = -kx
- 运动方程 d²x/dt² + ω²x = 0

#### 三部曲
1. **证简谐**: 证明 F = -kx 形式
2. **求ω**: ω = √(k/m) 或从方程读出
3. **定常数**: 由初始条件定 A, φ

### 波的干涉

#### 光程差分析
- 相位差 Δφ = (2π/λ) × 光程差
- 相长：光程差 = mλ
- 相消：光程差 = (m+½)λ

#### 驻波
- 波节：始终静止的点
- 波腹：振幅最大的点
- 相邻波节间距 = λ/2

## 5. 电磁场模式

### 电场分析

#### 高斯定理应用流程
1. **分析对称性**（球/柱/平面对称）
2. **选择高斯面**（与场强等距）
3. **计算电通量** ∮E·dA
4. **计算 enclosed charge**
5. **求解 E**

#### 常见Pattern
| Pattern | 高斯面 | 适用 |
|---------|--------|------|
| `场-球对称` | 同心球面 | 点电荷、均匀带电球 |
| `场-轴对称` | 同轴圆柱 | 无限长带电直线、圆柱 |
| `场-平面对称` | 圆柱/盒子 | 无限大带电平面 |

### 磁场分析

#### 安培环路定理
∮B·dl = μ₀I_enc

#### 常见Pattern
| Pattern | 安培环路 | 适用 |
|---------|----------|------|
| `场-长直导线` | 同心圆 | 无限长直电流 |
| `场-螺线管` | 矩形回路 | 长直螺线管内部 |
| `场-环形线圈` | 同心圆 | 螺绕环 |

### 电磁感应

#### 法拉第定律
ε = -dΦ/dt

#### 动生 vs 感生
| 类型 | 原因 | 公式 |
|------|------|------|
| 动生 | 导体运动 | ε = ∫(v×B)·dl |
| 感生 | 磁场变化 | ε = -∫(∂B/∂t)·dA |

## 6. 电路分析模式

### 基尔霍夫定律

#### 节点定律 (KCL)
ΣI_in = ΣI_out
- 流入节点的电流 = 流出节点的电流

#### 回路定律 (KVL)
Σε = ΣIR
- 沿闭合回路，电动势升 = 电势降

### 等效电路简化

#### 步骤
1. 合并串并联电阻
2. 处理对称性（等势点可短接）
3. Y-Δ变换（必要时）
4. 电源等效变换

### RC/RL暂态过程

#### 时间常数
- RC电路：τ = RC
- RL电路：τ = L/R

#### 充电/放电公式
- 充电：q = Q_max(1 - e^(-t/τ))
- 放电：q = Q_max e^(-t/τ)

## 7. 光学模式

### 干涉

#### 双缝干涉
- 条纹间距：Δx = λD/d
- 光程差：δ = d·sinθ ≈ d·x/D

#### 薄膜干涉
- 光程差：δ = 2nd + λ/2（半波损失）
- 相长/相消条件根据具体问题

### 衍射

#### 单缝衍射
- 暗纹：a·sinθ = mλ
- 半波带法分析

#### 光栅衍射
- 主极大：d·sinθ = mλ
- 缺级：当 d/a = 整数比时

## 8. 热学模式

### 热力学第一定律
ΔU = Q - W

#### 符号约定
- Q > 0: 吸热
- W > 0: 系统对外做功

### 常见过程

| 过程 | 特征 | Q | ΔU | W |
|------|------|---|----|---|
| 等容 | V = const | nCvΔT | nCvΔT | 0 |
| 等压 | p = const | nCpΔT | nCvΔT | pΔV |
| 等温 | T = const | nRTln(V₂/V₁) | 0 | nRTln(V₂/V₁) |
| 绝热 | Q = 0 | 0 | nCvΔT | -ΔU |

### 循环过程
- 顺时针（热机）：吸热→做功
- 逆时针（制冷）：做功→吸热
- 效率：η = W/Q_in = 1 - Q_out/Q_in

## 9. 近代物理模式

### 相对论

#### 近似条件
- v ≪ c: 用经典力学
- v ~ c: 必须用相对论

#### 常用公式
- 时间膨胀：Δt = γΔt₀
- 长度收缩：L = L₀/γ
- 质能关系：E = γmc²

### 量子力学基础

#### 量子化条件
- 能量量子化：E_n = n·hf 或特定形式
- 角动量量子化：L = n·ℏ

#### 波函数性质
- 归一化：∫|ψ|²dV = 1
- 概率密度：|ψ|²

## 专项训练规则

### 微图解训练

#### 每日练习 (10分钟)
1. 随机选一道题
2. 只用Scene/FBD/Field/Loop之一
3. 限时30秒画出
4. 检查：信息是否完整？是否清晰？

### Pattern识别训练

#### 步骤
1. 看到题目先猜Pattern（不说具体数值）
2. 验证Pattern是否适用
3. 按Pattern套路解题
4. 记录：Pattern匹配度评分（1-5）

### 错题深度复盘

#### C类错误（完全没思路）
1. 重新读题，自己尝试
2. 对比答案，找卡点
3. 建立Pattern卡片
4. 找3道同类题练习

## 完美主义拦截器（物理版）

### 物理特有卡点信号
- "微图解画了好久还不满意" → 超过60秒就停
- "不知道选哪个原理" → 先假设一个，验证
- "近似条件不确定" → 列出两种极端情况对比
- "代数太复杂算不下去" → 检查是否有更简单的参考系

### 推荐策略
1. **物理直觉优先**: 先估算数量级，再精确计算
2. **极限检查法**: 用极端情况快速验证合理性
3. **对称性利用**: 看看能不能减少计算量
4. **能量角度**: 复杂力学问题换个角度用能量
