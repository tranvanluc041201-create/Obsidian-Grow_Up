## 📂 文件结构

每个导师模式包含：

```
90_导师模式/[学科]/
├── 01_L1_CORE.md       # 速查卡 + 核心规则（必读）
├── 02_L2_MODES.md      # 详细模式规则（按需读取）
└── 03_L3_TEMPLATES.md  # 模板索引（整理笔记时用）
```

- [[C/01_L1_CORE|C 语言导师]] → `10_编程`
- [[EE/01_L1_CORE|电气导师]] → `10_电气`
- [[Math/01_L1_CORE|数学导师]] → `10_数学`

## 🎯 使用流程

### 日常学习
1. 说出**激活词** → AI 加载 L1 核心
2. 按需深入 → AI 加载 L2 详细规则（小测/高阶/止损）
3. 整理笔记 → 使用 L3 模板

### 整理笔记
1. 打开 `99_Templates/[学科]/`
2. 选择合适的模板
3. 复制到对应学科的学习目录
