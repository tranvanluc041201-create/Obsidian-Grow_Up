## 使用方式

1. **直接复制**：复制模板内容到新文件，按需修改
2. **Obsidian 模板插件**：将模板文件配置为 Obsidian 模板，用 `{{tp}}` 快速插入
3. **当前推荐顺序**：先用 `T1` 跑每日主流程，再按需补 `T4 / T6 / T7`
