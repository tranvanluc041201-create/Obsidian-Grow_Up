## 示例：罗尔定理应用

```markdown
---
type: proof-analysis
course: 高数
theorem: 罗尔定理
difficulty: L2
created: 2026-03-03
tags: [证明, 中值定理]
---

# 证明存在ξ使f'(ξ)=0

## 结论特征
- 结论形式：存在ξ使f'(ξ)=0
- 关键词："存在"、"某点"、"导数为0"

## 首选方法
**方法**：罗尔定理

### 方法前提
- [ ] f(x)在[a,b]连续
- [ ] f(x)在(a,b)可导
- [ ] f(a)=f(b)

### 辅助函数模板
如果题目不给f(a)=f(b)，需要构造：
```
F(x) = f(x) - [f(b)-f(a)]/(b-a) * (x-a) - f(a)
```
使得F(a)=F(b)=0

## 逆向-正向双链分析

### 逆向链
```
要证 f'(ξ)=0
  ← 罗尔定理
    ← 需要 f(a)=f(b)
```

### 正向链
```
已知条件
  → 验证连续性、可导性
    → 检查/构造f(a)=f(b)
```

### 交汇点
找到/构造出f(a)=f(b)即可应用罗尔定理

## 完整证明步骤
1. 验证f(x)满足罗尔定理前两个条件
2. 检查或构造f(a)=f(b)
3. 应用罗尔定理，得证

## 常见变式
| 变式 | 变化点 | 应对策略 |
|------|-------|---------|
| 变式1 | 不给f(a)=f(b) | 构造辅助函数 |
| 变式2 | 要证f'(ξ)=k | 转化为f'(ξ)-k=0 |
| 变式3 | 二阶导数 | 对f'(x)再用一次罗尔 |

## 易错点
- ❌ 忘记验证连续性和可导性
- ❌ 辅助函数构造错误
- ❌ 端点选择不当

## 链接相关证明题

- [[柯西中值定理证明]]
```

## 证明题套路速查表

| 结论特征 | 首选方法 | 辅助函数模板 |
|---------|---------|-------------|
| f'(ξ)=0 | 罗尔定理 | 直接找f(a)=f(b) |
| f'(ξ)=k | 拉格朗日 | [f(b)-f(a)]/(b-a) |
| f'(ξ)/g'(ξ)=... | 柯西 | 比值形式 |
| f''(ξ)=0 | 两次罗尔 | Rolle→Rolle |
| 存在两个不同点 | 两次中值定理 | 区间分割 |
