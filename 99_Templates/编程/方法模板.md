# 方法映射模板

*用于B类错题：方法不会用，建立可复用的解题模式*

---

## 快速复制模板

```markdown
---
type: method-map
course: 高数/概率论
method: 方法名称
created: <% tp.date.now("YYYY-MM-DD") %>
tags: [方法, 章节]
---

# 方法：{{方法名}}

## 适用信号（什么情况下用）
- [ ] 信号1：...
- [ ] 信号2：...
- [ ] 信号3：...

## 核心决策树（你的卡点）
```
决策起点？
  → 分支条件1 → 处理方法1
  → 分支条件2 → 处理方法2
  → 分支条件3 → 处理方法3
```

## 易错点检查清单
- [ ] 检查点1：...
- [ ] 检查点2：...
- [ ] 检查点3：...

## 链接到具体错题
- [[错题1]] - 错误类型1
- [[错题2]] - 错误类型2

## 变式题库（链接）
- [ ] 基础：[[错题1]]
- [ ] 进阶：[[错题2]]
- [ ] 综合：[[错题3]]
```

## 示例：泰勒展开

```markdown
---
type: method-map
course: 高数
method: 泰勒展开
created: 2026-03-03
tags: [方法, 极限]
---

# 方法：泰勒展开

## 适用信号
- [ ] 0/0型极限，分子分母是混合函数
- [ ] 需要比较函数的局部行为
- [ ] 求高阶导数

## 核心决策树
```
需要展开？
  → 看分母阶数 → 展开到同阶
  → 比较无穷小阶数 → 展开到最低次幂为n
  → 误差估计 → 用余项公式
```

## 易错点检查清单
- [ ] 展开阶数够了吗？（用分子分母同阶验证）
- [ ] 佩亚诺余项 vs 拉格朗日余项的选择
- [ ] 展开点选对了吗？（通常x₀=0）

## 链接到具体错题
- [[错题1]] - 展开阶数不够
- [[错题2]] - 余项选择错误

## 变式题库
- [ ] 基础：[[极限计算-基础]]
- [ ] 进阶：[[极限计算-进阶]]
- [ ] 综合：[[高阶导数-综合]]
```

## 使用说明

### 何时建立方法映射卡
- 同一方法出现≥2次错误
- 方法会但不知道怎么用（信号识别弱）
- 需要整理决策树（步骤多、条件复杂）

### 复习方式
1. **每周五晚**：看一遍本周建立的方法卡
2. **遇到新题**：先看方法卡，再做题
3. **SR复习**：只复习决策树部分
