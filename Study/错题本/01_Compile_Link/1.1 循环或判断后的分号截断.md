### [1.1] 循环/判断后的分号截断
* **标签**：#Syntax #ControlFlow
* **场景**：写 `for` 循环或 `if` 语句时手滑多打了个分号。
* **错误代码**：
    ```c
    for(int i=0; i<10; i++); // ❌ 这里的 ; 结束了循环体
    {
        printf("Running...\n"); // 结果：只打印一次，而不是10次
    }
    ```
* **根因 (Root Cause)**：
    > **空语句 (Empty Statement)**。
    > C 语言允许空语句。分号表示“什么都不做”。循环体执行了 10 次“空操作”，循环结束后，才顺序执行下面的代码块。
    运用:用于i++十次
* **修正 (Fix)**：
    > 删掉 `)` 后面的分号。
