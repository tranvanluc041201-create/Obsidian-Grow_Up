### [1.2] 宏定义的运算优先级
* **标签**：#Macro #Preprocessor
* **场景**：定义宏常量进行计算。
* **错误代码**：
    ```c
    #define A 2
    #define B A+3
    int result = B * 5; // ❌ 预期 (2+3)*5=25，实际 2+3*5=17
    ```
* **根因 (Root Cause)**：
    > **纯文本替换**。
    > 宏不是函数，预处理阶段只是简单地把 `B` 替换为 `2+3`,并不会进行运算。代码变成了 `int result = 2+3 * 5;`。
* **修正 (Fix)**：
    ```c
    #define B (A+3) // 永远给宏表达式加括号！
    ```
