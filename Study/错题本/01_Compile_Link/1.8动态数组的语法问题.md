### 2. 错误代码 (Error)
```c
// #include <stdio.h>

int main() 
{
int odd=0,even=0,n=0,a[n],i;
for(n=1,a[0]=1; a[n-1]!=-1; n++)
{
scanf("%d",&a[n]);
}
for(i=1; i<n-1; i++)
{ if(a[i]%2==0)
even++;
else
odd++;
}
printf("%d %d",odd,even);

return 0;
}
```

### 3. 根因 (Root Cause)
>如果n未初始化,会导致a[n]出错,导致程序崩溃或者内存崩溃
而n一旦赋值,a[n]就确定了,后面n改变a[n]也不会改变
