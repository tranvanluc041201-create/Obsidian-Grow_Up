### 【题目】Q8 指针定义语法的视力测试


题目：下列定义语句中正确的是？
A. char a='A' b='B';
B. float *a = b = 10.0;
C. int a=10, *b=&a;
D. float *a, b=&a;

【你的作答区】

你的选项：[ C  ]
正确答案：C

☠️ 你的思维误区：
这种题考的不是逻辑，是眼睛和基本语法规则。

🧠 逻辑修正：

A: a='A' 和 b='B' 中间缺了逗号。Syntax Error。

B: float *a 初始化时 b 还未定义。

C (正确)：int a=10 定义变量；*b=&a 定义指针并指向 a。逻辑通顺。

D: float *a, b=&a; 这里 b 被定义为 float (非指针)，却试图赋给它地址 &a。类型不匹配。
