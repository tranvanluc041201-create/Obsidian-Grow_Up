### 【题目】Q3 结构体传参的“复印件”效应


代码片段：

struct student { int id; float score; };
void modify(struct student s) {
    s.id = 9999;
}
int main() {
    struct student stu = {1001, 85.0};
    modify(stu);
    printf("%d", stu.id);
}


问题：输出结果是？
A. 9999
B. 1001
C. 编译错误
D. 随机值

在 C 语言中，除非你传指针，否则所有参数都是Pass by Value (值传递)。
modify 函数里拿到的 s，只是 main 函数里 stu 的一份复印件。
