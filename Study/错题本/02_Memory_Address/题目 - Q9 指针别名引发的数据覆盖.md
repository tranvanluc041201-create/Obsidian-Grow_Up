### 【题目】Q2 指针别名引发的数据覆盖


代码片段：

#include <stdio.h>
int main(void)
int a=3,b=5,x=10,y=20,t,*p;
p=&a,
if(a<b){*p= a,a= b, b= *p;}
if(x< y){t = x; x=y y=t;}
printf("\n %d,%d,%d,%d", a, b, x, y);
return 0;

A. 5,3,20,10
B. 3,5,10,20
C. 3,3,20,10
D. 5,5,20,10

【你的作答区】
a<b;->*p= a
注意:此处交换值有问题,不要看到互文就判断正确,还要看开头是否正确
此处将a的值赋给了a


你的选项：[ D  ]

正确答案：D

☠️ 你的思维误区：
你以为 *p 是个临时的保险箱，存了 a 的旧值？

🧠 逻辑修正：
p 是 a 的地址，*p 就是 a 本身（别名）。

*p = a 等于 a = a。废话操作，什么都没存下来。

a = b。a 变成了 5。注意！此时 *p 也自动变成了 5。

b = *p。读 *p (即 5)，赋给 b。b 还是 5。
结果：a 的旧值 3 彻底丢失，两个数都变成了 5。
