### 【题目】Q1 永远为真的数学谎言


代码片段：

int x = 3, y;
do {
    y = x--;
    if (!y) { printf("x"); continue; }
    printf("#");
} while (1 <= x <= 2);


问题：该程序的运行结果是？
A. 输出 ##
B. 输出 ##x
C. 死循环 (Infinite Loop)
D. 语法错误

【你的作答区】

你的选项：[   ]

你的预判：(例如：x如何变化...)

正确答案：C (死循环)

☠️ 你的思维误区：
你是不是以为 1 <= x <= 2 真的能限制 x 在 1 到 2 之间？

🧠 逻辑修正：
C 语言不像 Python。

表达式从左向右结合：先算 (1 <= x)。结果只能是 0 (假) 或 1 (真)。

然后算 (结果) <= 2。显然，0 和 1 永远小于等于 2。

结论：条件恒为真 (Always True)，循环永不停止。
