### 【题目】Q4 逻辑短路的冷酷规则


代码片段：

int x = 0, y = 5, z = 3;
while (z-- > 0 && ++x < 5) {
    y--;
}
// 循环结束后...
printf("x=%d, z=%d", x, z);


问题：输出结果是？
A. x=3, z=0
B. x=3, z=-1
C. x=4, z=-1
D. x=2, z=-1

正确答案：B (x=3, z=-1)

☠️ 你的思维误区：
你是不是在纠结最后一次 ++x 为什么没执行？或者 z 为什么是 -1？

🧠 逻辑修正：

关于 z：z-- 是后减。

第一次判断：3>0 (T), z变2,x==1,1<5

第二次判断：2>0 (T), z变1,x==2,2<5

第三次判断：1>0 (T), z变0,x==3,3<5

第四次判断：0>0 (False), z变-1。

关于 x (短路)：

在第四次判断时，因为 && 左边已经是 False，右边的 ++x 直接被屏蔽，完全不执行。

所以 x 停留在 3，没有变成 4。
