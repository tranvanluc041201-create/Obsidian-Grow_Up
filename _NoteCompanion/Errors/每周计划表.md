## 🎯 周六模考日详细流程（130min）

### 第一阶段：听力模考（60min）
- [ ] **听力整套A**（30min计时）
  - Section A：短对话
  - Section B：长对话  
  - Section C：听力篇章
- [ ] 记录：正确率 + 错因标签至少1个
- [ ] **听力整套B**（30min计时）
  - 讲座/讲话部分
- [ ] 记录：正确率 + 错因标签至少1个

### 第二阶段：阅读模考（40min）
- [ ] **阅读整套**（40min计时）
  - Passage 1
  - Passage 2
  - Passage 3
- [ ] 记录：完成时间 + 错因标签至少1个

### 第三阶段：写作翻译（30min×2）
- [ ] **写作**（30min计时）
  - 按模板写1篇作文
  - AI纠错：只要1个关键语法问题 + 2个可复用表达
- [ ] **翻译**（30min计时）  
  - 5-6句句式转换练习
  - AI纠错同上

## 📊 周日复盘流程（10min）

**使用模板**：`T7【周复盘】`

**4指标检查**：
1. [ ] 听力错点数量趋势
2. [ ] 听力整套正确率趋势
3. [ ] 阅读完成时间趋势
4. [ ] 断更天数（含MVP日也算不断更）

**输出**：本周Top3问题 + 下周微调

## 情绪降级规则

当 `mood=annoyed` 或 `fatigue=yes` 时：

**早读40** → 保持正常（不能降级，基础要保住）

**晚间60-75** → 降级为轻量版（30min）：
- 错点回放20
- 真题检验10
- 其余可选，不强制

**副轨** → 强制S1难度，10min上限
