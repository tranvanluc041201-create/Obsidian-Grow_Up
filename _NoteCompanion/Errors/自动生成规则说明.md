# 自动生成规则说明 + 失败回退流程

---

## 🤖 自动生成明日页（首选方式）

### 触发条件
每天完成今日页后，系统自动根据今日数据生成明天的任务页。

### 输入数据（来自今日页 YAML）
```yaml
date: 2025-03-03
main_set_id: CET6_TBD_001
main_listening_material_id: CET6_TBD_001_Listening_SectionA
main_reading_material_id: CET6_TBD_001_Reading_P1
listening_error_points: 
  - "time_or_keyword: xxx / 原文片段: xxx"
  - "time_or_keyword: yyy / 原文片段: yyy"
  - "time_or_keyword: zzz / 原文片段: zzz"
reading_error_tags: [定位, 同义替换]
mood: ok
fatigue: no
```

### 输出内容（明日页）
1. **明日早读**：默认推进"同一套真题的下一段/下一篇"
   - 如果无法自动识别下一段：沿用同ID并在标题写"NEXT SEGMENT"
   
2. **明日晚间**：自动生成"错点回放清单"
   - 每条错点包含：听3→看→听2→检验标准
   
3. **复习提醒**：不背单词只做今日复习；新增≤12

4. **技术泛听候选**：AI公司动态（过去7天不重复）
   - 格式：`候选标题/关键词3-5/难度S1或S2/时长10-15`

### 降级规则
- 若 `mood=annoyed` 或 `fatigue=yes`：
  - 副轨强制S1难度
  - 主轨晚间自动降级为"轻量版"（错点回放20 + 检验10 = 30min）

## 🔧 失败回退流程（手动方式）

**如果无法自动生成明日页，按以下步骤手动操作**：

### 步骤（≤5步）

1. **复制今日页的 listening_error_points 三条**
   - 打开今日任务页
   - 复制 YAML 中的 `listening_error_points` 列表

2. **粘贴到明日页 listening_error_points**
   - 创建新文件：`/01_Daily/YYYY-MM-DD.md`
   - 粘贴复制的错点数据

3. **设置 mood / fatigue 字段**
   - 根据今日状态设置：`mood: ok/annoyed/tired`
   - 设置：`fatigue: no/yes`

4. **在同页生成回放清单与副轨候选**
   - 复制模板 T3【晚间60-75】中的错点回放部分
   - 填入今日3条错点
   - 添加副轨候选（从 TECH_FEED_SOURCES 选一个）

5. **保存为明日日期文件名**
   - 文件名格式：`YYYY-MM-DD.md`
   - 示例：`2025-03-04.md`
