# 资料整理检查清单

> 📚 最少需要：8-12套近年 CET-6 真题（PDF+听力音频+解析）

---

## 资料获取任务

### 1. 获取真题资料
- [ ] 购买/下载 8-12套 CET-6 真题
- [ ] 确保包含：PDF试卷 + MP3音频 + 答案解析
- [ ] 推荐来源：星火英语/新东方 CET-6 真题集

### 2. 按命名规则整理
**命名格式**：
```
CET6_YYYYMM_SetNN/
├── Listening.mp3              # 听力音频
├── Transcript.md              # 听力原文/解析
├── Reading.pdf                # 阅读真题
├── Answer_Key.pdf             # 答案解析
└── Writing_Translation.pdf    # 写作翻译真题
```

**示例**：
```
CET6_202312_Set01/
├── Listening.mp3
├── Transcript.md
├── Reading.pdf
├── Answer_Key.pdf
└── Writing_Translation.pdf
```

### 3. 放置到正确路径
- [ ] 创建目录：`/10_English/11_CET6/_assets/`
- [ ] 每套真题一个文件夹：`CET6_YYYYMM_SetNN/`
- [ ] 将整理好的文件放入对应文件夹

### 4. 更新索引页
- [ ] 打开 `/10_English/11_CET6/_assets/index/CET6_INDEX.md`
- [ ] 添加套号信息到表格
- [ ] 标记状态为"未做"（🔴）

### 5. 准备微信传输包
- [ ] 创建目录：`/10_English/11_CET6/_assets/_wechat_pack/`
- [ ] 每周复制本周要听的MP3到该目录
- [ ] 按周整理，避免一次性传太多

---

## 微信传输音频流程

**电脑→微信→平板→笔记软件**（≤6步）：

- [ ] **Step1**：在电脑打开 `/10_English/11_CET6/_assets/_wechat_pack/`
- [ ] **Step2**：选中要传的 `.mp3` 文件 → 右键 → "发送到" → "微信文件传输助手"
- [ ] **Step3**：在平板上打开微信 → 文件传输助手
- [ ] **Step4**：点击音频文件 → "用其他应用打开" → 选择你的笔记软件
- [ ] **Step5**：在笔记软件里创建文件夹 `/CET6_听力/`
- [ ] **Step6**：保存音频，命名为 `CET6_SetNN_SectionX.mp3`
