## 🔧 技术实现方案

### 自动获取B站视频信息的完整流程

```
1. 搜索UP主最新视频
   API: https://api.bilibili.com/x/web-interface/search/all?keyword=UP主名
   返回: BV号、视频列表

2. 获取视频详细信息
   API: https://api.bilibili.com/x/web-interface/view?bvid=BV号
   返回: 标题、描述、时长、发布时间、UP主信息

3. 提取CID
   从视频详情中获取cid参数

4. 获取弹幕
   API: https://api.bilibili.com/x/v1/dm/list.so?oid=cid
   返回: 弹幕列表（XML格式）

5. 获取字幕（可选）
   API: https://api.bilibili.com/x/player/wbi/v2/closedcaption?bvid=BV号
   返回: CC字幕内容（如视频有）
```

### PowerShell命令示例

```powershell
# 1. 搜索视频
$headers = @{"User-Agent"="Mozilla/5.0"}
$search = Invoke-WebRequest -Uri "https://api.bilibili.com/x/web-interface/search/all?keyword=橘鸦Juya AI早报" -Headers $headers
$bvid = ($search.Content | ConvertFrom-Json).data.result.video[0].bvid

# 2. 获取视频详情
$video = Invoke-WebRequest -Uri "https://api.bilibili.com/x/web-interface/view?bvid=$bvid" -Headers $headers
$cid = ($video.Content | ConvertFrom-Json).data.cid

# 3. 获取弹幕
$danmaku = Invoke-WebRequest -Uri "https://api.bilibili.com/x/v1/dm/list.so?oid=$cid" -Headers $headers
```
