## 🎯 Problem（问题）

悬挂Else问题：

```c
int x=2, y=-1, z=2;
if(x < y)
    if(y < 0) z=0;
    else z+=1;    // else属于哪个if？
printf("%d", z);  // 输出？
```

**答案**：2（else匹配的是内层if，但外层if条件不满足，整个代码块被跳过）

---

## 💥 发生了什么

**C编译器不看缩进**：
```c
// 实际逻辑
if (x < y) {        // 2 < -1 为 False
    if (y < 0) z=0; // 整个代码块被跳过
    else z+=1;       // else属于内层if
}
// z根本没被动过，还是2
```

---

## 🤔 当时我是怎么想的

> "else应该和外层if配对..."> "缩进表明else属于外层if..."> 
> **盲区**：不知道else总是匹配**最近的未配对**的if。

---

## 🔍 Analysis（分析）

### 配对规则
```c
if (a)
    if (b)
        x = 1;
    else        // 和最近的if(b)配对
        x = 2;

// 如果希望else和if(a)配对，用花括号
if (a) {
    if (b)
        x = 1;
} else {
    x = 2;
}
```

---

## ✅ Refix（修正）

```c
// ✅ 始终使用花括号
if (x < y) {
    if (y < 0) {
        z = 0;
    } else {
        z += 1;
    }
}
```
