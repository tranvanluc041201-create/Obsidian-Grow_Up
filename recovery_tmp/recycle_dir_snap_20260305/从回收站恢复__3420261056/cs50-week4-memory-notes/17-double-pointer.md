## 🎯 核心概念

### 什么是指针的指针？

```c
int x = 10;
int *p = &x;      // p 指向 x（存x的地址）
int **pp = &p;    // pp 指向 p（存p的地址）
```

**类型分解**：
| 变量 | 类型 | 存的值 | 含义 |
|------|------|--------|------|
| x | int | 10 | 整数 |
| p | int* | &x | x的地址 |
| pp | int** | &p | p的地址 |

---

## 📐 内存图解

```
地址从高到低（示意）：

┌─────────────────────────────────────┐
│  pp (int**)                         │
│  地址: 0x1000                       │
│  值: 0x500 ──────┐                  │
└──────────────────┼──────────────────┘
                   │
                   ▼
┌─────────────────────────────────────┐
│  p (int*)                           │
│  地址: 0x500                        │
│  值: 0x2000 ─────┐                  │
└──────────────────┼──────────────────┘
                   │
                   ▼
┌─────────────────────────────────────┐
│  x (int)                            │
│  地址: 0x2000                       │
│  值: 10                             │
└─────────────────────────────────────┘
```

**访问路径**：
```
pp (0x1000) 
    → *pp = p的值 = 0x500
    → **pp = *p = x = 10
```

## 🔍 表达式解析

```c
int x = 10;
int *p = &x;
int **pp = &p;
```

| 表达式 | 等价写法 | 结果 | 类型 |
|--------|---------|------|------|
| `*pp` | `pp[0]` | &x (p的值) | int* |
| `**pp` | `*pp[0]` 或 `pp[0][0]` | 10 (x的值) | int |
| `*p` | `p[0]` | 10 (x的值) | int |

**注意**：`**pp = *p` 这个说法是错误的！
- `**pp` 的结果是 `10`（int）
- `*p` 的结果也是 `10`（int）
- 两个表达式**值相等**，但不是同一个东西

正确说法：**pp 解引用两次得到的是 p 解引用一次得到的值**

## 💡 数组视角理解

### 指针数组

```c
int *arr[3];      // 指针数组：3个 int* 元素
arr[0] = &x;
arr[1] = &y;
arr[2] = &z;

int **pp = arr;   // pp 指向数组首元素

// 以下等价：
pp[0]     // 第0个指针 = &x
pp[1]     // 第1个指针 = &y
pp[0][0]  // 第0个指针指向的值 = x
pp[1][0]  // 第1个指针指向的值 = y
```

### 二维数组的联系

```c
// 静态二维数组
int arr[3][4];
int (*p)[4] = arr;  // p 是数组指针

// 动态二维数组（指针数组方式）
int **a = malloc(3 * sizeof(int*));
for (int i = 0; i < 3; i++) {
    a[i] = malloc(4 * sizeof(int));
}
// a[i][j] 访问
```

**关键区别**：
- 静态：`arr[i][j]` 是一次计算偏移（连续内存）
- 动态：`a[i][j]` 是两次寻址（先找行，再找列）

## 📝 代码跟踪练习

```c
int a = 1, b = 2, c = 3;
int *p1 = &a, *p2 = &b, *p3 = &c;
int *arr[3] = {p1, p2, p3};
int **pp = arr;

// 以下表达式的值：
*pp        // ?
**pp       // ?
*(pp + 1)  // ?
**(pp + 1) // ?
pp[2][0]   // ?
```

**答案**：
- `*pp` = &a (p1的值)
- `**pp` = 1 (a的值)
- `*(pp + 1)` = &b (p2的值)
- `**(pp + 1)` = 2 (b的值)
- `pp[2][0]` = 3 (c的值)

## 🎯 为什么需要二级指针？

### 场景1：动态二维数组

```c
int **matrix = malloc(rows * sizeof(int*));
for (int i = 0; i < rows; i++) {
    matrix[i] = malloc(cols * sizeof(int));
}
// matrix[i][j] 访问元素
```

### 场景2：函数修改指针本身

```c
void allocate(int **pp, int size) {
    *pp = malloc(size * sizeof(int));  // 修改调用者的指针
}

int main() {
    int *p = NULL;
    allocate(&p, 10);  // p 现在指向分配的内存
    free(p);
    return 0;
}
```

### 场景3：字符串数组（指针数组）

```c
char *names[] = {"Alice", "Bob", "Charlie"};
char **pp = names;

// names[0] 是 char*，指向 "Alice"
// pp[0][0] 是 'A'
```

## 💡 记忆口诀

> **指针数组用 **，动态二维要熟悉**

## ⚠️ 常见错误

| 错误 | 说明 |
|------|------|
| `**pp = *p` | 表达式比较，不是类型说明 |
| 忘记初始化 | `int **pp; **pp = 10;` 野指针崩溃 |
| 混淆层级 | `*pp` 是指针，`**pp` 才是值 |
