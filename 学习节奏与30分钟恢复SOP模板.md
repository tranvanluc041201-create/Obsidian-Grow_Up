## 什么时候必须进 30 分钟恢复
- [ ] 已连续完成 2 个深度学习块
- [ ] 同时出现 2 个以上预警信号
- [ ] 开始反复读，但吸收明显下降
- [ ] 身体紧绷已经明显影响理解力

## 预警信号
- [ ] 眼睛发酸
- [ ] 脖子肩膀发紧
- [ ] 开始反复读同一句
- [ ] 想摸手机
- [ ] 明明在看，但脑子不进东西

## 30 分钟恢复流程

### 1. 闭眼躺 10 分钟
- [ ] 不刷手机
- [ ] 只定闹钟
- [ ] 不强求睡着

### 2. 小范围整理 8 分钟
- [ ] 只选一个范围：桌面 / 床铺 / 地面 / 书包
- [ ] 只做收纳，不扩成大扫除

### 3. 和舍友正常聊 10 分钟
- [ ] 聊生活或轻松内容
- [ ] 不聊比较、焦虑、任务压力
- [ ] 到点收住，不继续拖长

### 4. 剩余 2 分钟
- [ ] 喝水 / 洗脸 / 站到窗边
- [ ] 看一眼“休息后下一步”

## 恢复后缓启动（5 分钟）
- [ ] 先看一眼卡点
- [ ] 只做一个最小动作：重读题目 / 看上一段代码 / 写第一步
- [ ] 不要休完立刻冲最难部分

## 止损点
- 一个学习块里切了另一门课
- 整理超过 10 分钟，还在不断加任务
- 聊天明显失控，已经拖到 20 分钟以上
- 休息后比休息前更烦躁、更空、更不想继续

## 恢复后自检
- [ ] 脑子是否比刚休息前更清一点
- [ ] 情绪是否更稳一点
- [ ] 我是否知道接下来第一步做什么

> [!summary]
> 如果上面 3 项里有 2 项变好，这次恢复就是有效的。
